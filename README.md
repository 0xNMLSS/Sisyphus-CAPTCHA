# Sisyphus CAPTCHA

A single-page, slider-based “CAPTCHA” toy: drag the handle to push the boulder uphill, deal with rollback, and verify. **This is not a secure or production-ready CAPTCHA**—it is a thematic mini-game and demo.

## Run locally

1. Place all assets in the same folder as `Sisyphus.html` (see [Assets](#assets)).
2. Open `Sisyphus.html` in a modern desktop or mobile browser (double-click or use a local static server if you prefer).

No build step or package manager is required.

## Assets

Expected alongside `Sisyphus.html`:

| File | Role |
|------|------|
| `Background.png` | Game area background |
| `Stone.png` | Boulder sprite |
| `Sisyphus.png` | Character sprite |
| `bgm.aac` | Optional loop background music (muted by default until interaction) |

If an asset is missing, the page may still load but images or audio will not appear or play.
